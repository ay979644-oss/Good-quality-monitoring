## Food Quality Monitoring System 🍔

A simple web-based AI project to check food quality using image upload.

## Features
- Upload food image
- Detect Fresh / Stale / Unsafe
- Quality score (0–100)
- Simple UI

## Tech Stack
- Python (Flask)
- HTML, CSS
- Basic AI logic (Demo)

## How to Run

```bash
pip install -r requirements.txt
python app.py
